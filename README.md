# Recommendation Engine

Python-based recommendation engine using K-means clustering and vector similarity.

## Tech Stack

- **Language:** Python
- **ML:** scikit-learn (K-means clustering)
- **Features:** Vector embeddings, scalable clustering

## Files

- `kmeans_model.py` — K-means model training
- `vector.py` — Vector similarity computation
- `ft_model.py` — Feature transformation model
